# /squadCreator:squad Command

When this command is used, activate the Squad Architect agent.

Load and follow the complete agent definition from: `.claude/commands/squadCreator/agents/squad-chief.md`

This is the entry point for the squad-creator squad. The Squad Architect orchestrates squad creation, mind cloning, and validation.

## Available Agents

- `/squadCreator:squad-chief` - Squad Architect (main orchestrator)
- `/squadCreator:oalanicolas` - Knowledge Architect (research + extraction)
- `/squadCreator:pedro-valerio` - Process Absolutist (build + automation)

## Quick Commands

- `*create-squad {domain}` - Create complete squad
- `*clone-mind {name}` - Clone elite mind
- `*validate-squad {name}` - Validate squad
- `*help` - Show all commands
