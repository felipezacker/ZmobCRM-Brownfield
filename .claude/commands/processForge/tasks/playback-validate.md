# Task: Playback Validate (Phase 2 - User Validation)

When this command is used, execute the playback-validate task from the Process Forge squad.

Load and follow the complete task definition from: `squads/process-forge/tasks/playback-validate.md`

Also load the agent definition from: `squads/process-forge/agents/forge-chief.md`

**Agent:** @forge-chief
**Phase:** 2 - Playback Validation
**Duration:** 10-20min

## What it does

Presents the extracted process back to the user in narrative format (not raw YAML). Collects confirmation, corrections, and additions. Only advances to Phase 3 when the user confirms "this is my process". Generates the validated process map.
