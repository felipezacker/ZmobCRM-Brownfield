# Task: Validate Squad (Phase 5 - Final Validation)

When this command is used, execute the validate-squad task from the Process Forge squad.

Load and follow the complete task definition from: `squads/process-forge/tasks/validate-squad.md`

Also load the agent definition from: `squads/process-forge/agents/forge-chief.md`

**Agent:** @forge-chief
**Phase:** 5 - Validation
**Duration:** 10-20min

## What it does

Final validation of the generated squad: structural validation (squad-validator.js), smoke tests (2/3 must pass), process coverage check (every PU-STEP mapped), and user approval. Produces the validation report.
