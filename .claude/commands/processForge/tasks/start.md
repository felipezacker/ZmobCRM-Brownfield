# Task: Start (Phase 0 - Setup)

When this command is used, execute the start task from the Process Forge squad.

Load and follow the complete task definition from: `squads/process-forge/tasks/start.md`

Also load the agent definition from: `squads/process-forge/agents/forge-chief.md`

**Agent:** @forge-chief
**Phase:** 0 - Setup
**Duration:** 3-5min

## What it does

Initializes the Process Forge pipeline: activates the Forge Chief, collects process name and scope from the user, creates the output directory structure, and hands off to the extraction phase.
