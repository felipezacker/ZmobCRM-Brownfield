# Process Forge Squad - Installed Commands

Slash prefix: `processForge`

## Available Agents

- `/processForge:forge-chief` - Pipeline Orchestrator (playback validation, quality gates)
- `/processForge:process-archaeologist` - Process Extraction Specialist (8-lens iterative extraction)
- `/processForge:forge-smith` - AIOS Squad Builder (architecture + assembly)

## Available Tasks

- `/processForge:start` - Setup pipeline and collect process info
- `/processForge:extract-process` - Deep iterative process extraction (8 lenses)
- `/processForge:playback-validate` - Present process back to user for validation
- `/processForge:architect-squad` - Decompose process into squad architecture
- `/processForge:assemble-squad` - Generate all AIOS squad artifacts
- `/processForge:validate-squad` - Structural validation + smoke tests
