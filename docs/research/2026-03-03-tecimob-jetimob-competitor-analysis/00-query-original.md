# Original Research Query

## Question

Research the property management features of Tecimob and Jetimob (Brazilian real estate CRMs) focusing on:

1. How they handle property listings (cadastro de imoveis)
2. How they match properties to contacts/leads
3. How they integrate with Orulo
4. What the property detail page looks like (fields, layout)
5. How they handle property photos
6. How deals/negotiations are linked to properties

## Context

- **Project:** ZmobCRM (brownfield CRM for real estate)
- **Competitors:** Tecimob (tecimob.com.br), Jetimob (jetimob.com)
- **Objective:** Identify practical features that ZmobCRM should implement to compete
- **Inferred Focus:** Product feature comparison, UX patterns, integration architecture
