# Query Original

## Pergunta do Usuario

Pesquisa aprofundada sobre o Beotto (beotto.com.br ou beotto.com), plataforma usada pela Melnick (incorporadora gaucha de alto padrao) para seus corretores visualizarem imoveis/empreendimentos.

## Topicos Solicitados

1. O que e o Beotto -- portal de vendas, CRM, catalogo para incorporadoras? Publico-alvo?
2. Como funciona para corretores -- acesso, visualizacao de empreendimentos, experiencia de uso
3. Features principais -- catalogo, espelho de vendas, tabela de precos, plantas, tours, simulador
4. Fluxo de venda -- corretor ve imovel, reserva, proposta, venda
5. Integracao com CRMs -- API, fluxo de dados
6. Comparacao com Orulo -- diferencas, complementaridade
7. Dados disponiveis para integracao -- empreendimentos, unidades, precos, fotos
8. Outras incorporadoras que usam

## Contexto Inferido

- **Projeto:** ZmobCRM (CRM imobiliario para corretores/imobiliarias no RS)
- **Stack:** React/TypeScript, Supabase, Next.js
- **Objetivo provavel:** Avaliar viabilidade de integracao entre ZmobCRM e Beotto para que corretores usando o ZmobCRM tenham acesso a dados de empreendimentos da Melnick
- **Mercado:** Imobiliario, foco em Porto Alegre / Rio Grande do Sul
- **Foco:** Tecnico (integracao) + Estrategico (posicionamento)
