## Story

**As a** corretor imobiliario,
**I want** filtrar contatos por classificacao, temperatura, tipo PF/PJ e corretor responsavel, executar acoes em massa e exportar CSV,
**so that** eu possa gerenciar minha carteira de clientes de forma eficiente, segmentar contatos e extrair dados para uso externo.

## Acceptance Criteria

1. Filtro por `classification` existe (select multi-opcao: COMPRADOR, VENDEDOR, LOCATARIO, LOCADOR, INVESTIDOR, PERMUTANTE)
2. Filtro por `temperature` existe (select: HOT, WARM, COLD, TODOS)
3. Filtro por `contact_type` existe (select: PF, PJ, TODOS)
4. Filtro por `owner` existe (select com lista de corretores da organizacao)
5. Filtro por `source` existe (select: WEBSITE, LINKEDIN, REFERRAL, MANUAL, TODOS)
6. Coluna "Responsavel" (owner) exibida na tabela com nome/avatar do corretor
7. Coluna "Origem" (source) exibida na tabela com badge
8. Filtros persistem na URL como query params (navegacao sem perder filtros)
9. Bulk actions toolbar aparece quando 1+ contatos selecionados: reatribuir owner, exportar selecionados
10. Reatribuir owner: modal com select de corretor → UPDATE contacts SET owner_id
11. Exportar CSV: gera arquivo com todos os campos visiveis dos contatos selecionados (ou todos se nenhum selecionado)
12. Sort funciona em todas as colunas novas (owner, source)
13. Contagem de resultados exibida ("X contatos encontrados")
